_ча̄ри-джанера пунах̣ пр̣тхак тина тина мӯрти  
кеш́ава̄ди йа̄ха̄ хаите вила̄сера пӯрти_

## Пословный перевод

_ча̄ри_\-_джанера_ — из первых четырех экспансий; _пунах̣_ — снова; _пр̣тхак_ — по отдельности; _тина_ _тина_ — по три из каждой; _мӯрти_ — формы; _кеш́ава_\-_а̄ди_ — начиная с Господа Кешавы; _йа̄ха̄_ _хаите_ — из которых; _вила̄сера_ _пӯрти_ — полностью проявляются экспансии _виласа_.

## Перевод

**«От каждой из этих четырех форм исходят еще по три формы, начиная с Кешавы. Так полностью проявляются виласа-рупы Господа».**
